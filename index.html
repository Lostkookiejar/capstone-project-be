Welcome to the API!

How to use this API:

  • GET /allreviews : Developer only endpoint. Returns a json payload with all entries in the database.
  • GET /reviews/user/:id : Returns a json payload with all reviews of a specific user, ordering it from the most recent to the least. User id should be passed as the paramter. In the frontend, that id is passed from firebase authentication uid.
  • GET /generate/review/:gameName : Passes a name of a video game as parameter. Returns a json payload with a Gemini written review of a game that should be slightly wrong.
  • POST /create/review/:user_id : Creates a new review with a user id passed as parameter. Request body should be an object with 'name', 'thumbnail', 'content', 'created_at', 'rating' and 'playtime' keys. All keys hold string values, except 'rating', which holds an integer value. Returns a payload with the same key-value pairings for immediate frontend state updates.
  • PUT /update/review/:id : Updates an existing review. The review's id is passed as a parameter, and the request body should contain an object with 'content', 'playtime', 'rating' and 'thumbnail' keys. The object is then returned as payload.
  • DELETE /delete/review/:id : Deletes an existing review. The review's id is passed as a parameter. Returns the review's id.
